## 1. Install

Use your favorite package manager to install Atelier® Kit.

<CodeGroup>

```sh npm
npm install --save @atlr/kit
```

```sh yarn
yarn add @atlr/kit
```

```sh pnpm
pnpm install @atlr/kit
```

</CodeGroup>
